# portfolio

Personal site for Diana Zhang — Finance @ Georgia State University, '28.

Built with Next.js 16 (App Router), TypeScript, Tailwind CSS v4, and Magic UI components. Deployed on Vercel.

## stack

- Next.js 16 (App Router, `src/`)
- TypeScript
- Tailwind CSS v4
- [Magic UI](https://magicui.design) — `animated-gradient-text`, `particles`, `border-beam`
- Inter via `next/font/google`
- `motion` for animations

## run locally

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## scripts

- `pnpm dev` — dev server
- `pnpm build` — production build
- `pnpm start` — run the built app
- `pnpm lint` — eslint
